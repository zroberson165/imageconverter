# Changelog

## [0.2.0] - 2026-03-06
### Added
- RAW file support (NEF, CR2, ARW) with automatic brightness correction
- `--recommend` flag to analyze images and suggest optimal settings
- `--recursive` flag to process subfolders
- `--resize` flag to resize images by percentage while maintaining aspect ratio
- File size summary after conversion

## [0.1.0]
### Added
- Initial release
- HEIC, JPG, PNG conversion
- Quality control via `--quality` flag
